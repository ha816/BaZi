# 후천 분석 (대운·세운) 기초 정리

## 선천 vs 후천

사주 분석은 크게 두 영역으로 나뉜다.

| 구분 | 선천 분석 (natal) | 후천 분석 (fortune) |
|------|------------------|-------------------|
| 데이터 | 태어난 순간의 팔자 | 시간의 흐름에 따른 간지 |
| 성격 | 고정값 (평생 불변) | 동적값 (시간에 따라 변함) |
| 내용 | 오행 분포, 일간 강약, 용신, 십신 | 대운, 세운 |
| 비유 | 체질 | 계절·날씨 |

## 대운 (大運) - 10년 단위의 큰 흐름

인생의 시기별 테마를 나타낸다. 계절에 비유할 수 있다.

### 계산 방법

1. **월주를 기점**으로 천간·지지를 하나씩 이동
2. **순행/역행** 방향은 성별 + 년주 천간의 음양으로 결정
   - 양남(양 천간 + 남자) / 음녀(음 천간 + 여자) → 순행 (+1)
   - 음남(음 천간 + 남자) / 양녀(양 천간 + 여자) → 역행 (-1)
3. **시작 나이**는 생일에서 가장 가까운 절(節)까지의 일수 ÷ 3
   - 순행이면 다음 절까지, 역행이면 이전 절까지
   - 1일 = 4개월, 3일 = 1년으로 환산

### 예시: 1990년 10월 10일 14:30 남자

- 년주: 庚(양) + 남자 → 순행
- 월주: 丙戌 기준으로 +1씩 전진
- 다음 절(입동)까지 28일 → 28 ÷ 3 ≈ 9세 시작

| 대운 | 나이 | 간지 |
|------|------|------|
| 1대운 | 9~18세 | 丁亥 |
| 2대운 | 19~28세 | 戊子 |
| 3대운 | 29~38세 | 己丑 |
| 4대운 | 39~48세 | 庚寅 |
| ... | ... | ... |

## 세운 (歲運) - 매년 바뀌는 운

그 해의 간지가 내 팔자에 미치는 영향이다. 날씨에 비유할 수 있다.

### 계산 방법

연도를 60갑자로 변환한 뒤, 천간·지지 각각의 십신을 본다.

### 예시: 일간이 己(토)인 사람의 2026년 세운

- 2026년 = 丙午
- 丙(화, 양) → 일간 己 기준 정인 (나를 생하는 오행, 다른 음양)
- 午(화, 양) → 일간 己 기준 정인
- 해석: 학문·자격증·어머니 기운이 강한 해

## 대운과 세운의 관계

| 대운 | 세운 | 해석 |
|------|------|------|
| 좋음 | 좋음 | 최고의 시기 |
| 좋음 | 나쁨 | 큰 흐름은 좋지만 그 해는 주의 |
| 나쁨 | 좋음 | 힘든 시기지만 그 해는 기회 |
| 나쁨 | 나쁨 | 가장 어려운 시기 |

대운이 인생의 큰 방향이라면, 세운은 그 안에서의 연도별 변화이다.
실제 감정에서는 대운의 간지와 세운의 간지를 모두 팔자와 대조하여 십신 관계, 상생/상극, 충/합 등을 종합적으로 본다.

## 코드에서의 표현

```python
from saju import natal, fortune

# 1. 선천 분석 (고정)
result = natal.full_analysis(["庚午", "丙戌", "己巳", "辛未"])

# 2. 후천 분석 (시간 변수)
fate = fortune.full_analysis(
    result.saju,
    year=2026,
    is_male=True,
    birth_year=1990, birth_month=10, birth_day=10,
    birth_hour=14, birth_minute=30,
)

fate.seun          # 세운: 2026년 간지의 십신
fate.daeun         # 대운: 10년 단위 간지 + 시작/끝 나이
fate.daeun[0]      # DaeunPeriod(ganji='丁亥', start_age=9, end_age=18)
```
