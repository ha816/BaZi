# 간지(干支) 기초 정리

## 구조

사주팔자(四柱八字)는 4개의 기둥(柱)으로 이루어지며, 각 기둥은 **천간(天干, stem)** + **지지(地支, branch)** 두 글자의 조합이다.

| 기둥 | 영문 | 의미 |
|------|------|------|
| 년주 | year_pillar | 태어난 해 |
| 월주 | month_pillar | 태어난 달 |
| 일주 | day_pillar | 태어난 날 |
| 시주 | hour_pillar | 태어난 시 |

## 천간(天干) - Heavenly Stems

10개: 甲 乙 丙 丁 戊 己 庚 辛 壬 癸

## 지지(地支) - Earthly Branches

12개: 子 丑 寅 卯 辰 巳 午 未 申 酉 戌 亥

## 60갑자(六十甲子)

천간 10개 × 지지 12개의 모든 조합은 120가지이지만, 실제로는 순서대로 하나씩 짝을 짓는 방식이므로 60가지만 존재한다.

- 甲子 → 乙丑 → 丙寅 → ... 순서로 천간과 지지를 동시에 한 칸씩 전진
- 홀수 번째 천간은 홀수 번째 지지하고만 조합
- 짝수 번째 천간은 짝수 번째 지지하고만 조합
- 홀수 천간 5개 × 홀수 지지 6개 = 30
- 짝수 천간 5개 × 짝수 지지 6개 = 30
- 합계: **60가지**

甲子로 시작해서 癸亥로 끝나면 한 바퀴 → 60년을 환갑(還甲)이라 한다.

## 코드에서의 표현

`sajupy`의 `cal_saju()` 반환값 기준:

```python
result["year_pillar"]   # "庚午" (천간 + 지지)
result["year_stem"]     # "庚"   (천간)
result["year_branch"]   # "午"   (지지)
# stem + branch == pillar
```

4개 기둥 × 2글자 = 8글자 → 이것이 팔자(八字)이다.
