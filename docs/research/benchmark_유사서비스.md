## 2. UX 패턴 비교

### 2-1. 무료/유료 콘텐츠 경계 설계

| 서비스 | 무료 공개 범위 | 유료 잠금 방식 | 전환 유도 CTA |
|--------|---------------|---------------|---------------|
| **점신** | 대부분 무료 (광고 기반) | 전화상담만 유료, 광고 강제 시청으로 콘텐츠 잠금 해제 | "3분 무료 상담" 체험 후 결제 유도 |
| **포스텔러** | 1,500개 무료 + 700개 유료 (전체의 ~30%) | 코인 단건 결제 (개당 ₩3,000~₩20,000) | 매일 무료 콘텐츠 1편 제공 → 더 보고싶으면 코인 구매 |
| **네이버운세** | 전체 무료 (연간 신년운세 기간 한정) | 유료 없음, 전문가 상담은 네이버 엑스퍼트 연동 | 검색 → 운세 → 엑스퍼트 상담 |
| **Co-Star** | 기본 차트 + 일일 메시지 | 대부분 유료, 개별 콘텐츠 $2.99~$11.99 | 블러 오버레이 + "Unlock" 버튼, 사용자 불만 高 |
| **The Pattern** | 기본 성격 분석 + 일일 업데이트 | "Personal Audio" 시리즈 구독 ($29.99/분기) | 콘텐츠 진입 시 맥락적 페이월, 연간/분기 플랜 제시 |
| **Sanctuary** | 일일 호roscope + 기본 차트 | Sanctuary+ ($20/월) | 5분 무료 전문가 리딩 체험 후 구독 전환 |

**주요 패턴 발견:**
- 성공한 서비스들은 "무료로 충분히 경험 → 더 깊이 알고 싶은 욕구 자극 → 유료 전환" 구조를 공통적으로 가짐
- Co-Star는 지나친 페이월로 사용자 불만 급증 (리뷰에서 반복 언급), 반면 The Pattern은 맥락적 페이월로 거부감 최소화
- 포스텔러는 코인 단건 구조로 구독 거부감 없이 소액 결제 유도

| 서비스 | 온보딩 단계 | 특징 |

## 6. 사주까치 적용 가능 인사이트 (우선순위 정렬)

### P0 — 즉시 적용 (핵심 차별화)

**[INS-1] "왜 이 점수인가" 근거 시각화 (= 점수 근거 툴팁)**
- 벤치마크 근거: The Pattern의 레이어드 정보 구조 (카드 요약 → 탭으로 상세 → 오디오)
- 사주까치 적용: `domain_scores`에 `reason: str` 추가, 각 영역 점수 옆에 아코디언/툴팁으로 "재물운 80점인 이유: 대운 정재 입고, 일지 합(合) 발생"
- 임팩트: 신뢰도 급상승, "와, 진짜 내 얘기네" 공감도 핵심 레버

**[INS-2] 까치 알림의 바이럴 설계 (Co-Star 벤치마크)**
- 벤치마크 근거: Co-Star의 엣지 있는 일일 알림이 밈화 → 30M 가입자
- 사주까치 적용: 오늘의 운세 알림을 단순 "오늘 재물운 좋아요"에서 벗어나 **사주까치 특유의 한국적 센스 있는 문장**으로 설계 (예: "까치가 울었으니 반가운 소식 올 날이야, 근데 지갑은 조심해")
- 임팩트: 공유 유도 → 바이럴 유입

**[INS-3] 맥락적 페이월 (The Pattern 벤치마크)**
- 벤치마크 근거: The Pattern은 "잠긴 챕터에 접근할 때" 페이월 표시 → Co-Star처럼 선제적 블러보다 거부감 낮음
- 사주까치 현황: `/analysis` 페이지 블러 CTA → `/analysis/deep` 유도
- 적용 방향: 블러 영역에 "이 분석을 읽다 보니 궁금한 게 생겼죠? 까치가 더 알려드릴게요" 식의 문구로 자연스럽게 전환
- 임팩트: 이탈률 감소, 무료→유료 전환율 향상

### P1 — 단기 적용 (리텐션 강화)

**[INS-4] 오늘의 운세를 매일 갱신되는 핵심 훅으로 강화**
- 벤치마크 근거: 점신(매일 오늘의 운세), 포스텔러(자정 피크), Co-Star(매일 알림)
- 사주까치 적용: 현재 `/daily` 엔드포인트 존재 → 알림 설정 + 일일 운세 변동 하이라이트 ("어제보다 재물운 +15점 상승")
- 임팩트: DAU 직결

**[INS-5] 피드백 루프 (해석 만족도 수집)**
- 벤치마크 근거: 점신 사용자 만족도 38%에 불과 (VoC 분석), 포스텔러는 콘텐츠 품질 개선으로 흑자 전환
- 사주까치 적용: 각 해석 카드 하단 "이 해석이 잘 맞나요? 👍 / 👎" → DB 누적 → Interpreter별 만족도 측정 → 낮은 것부터 우선 개선
- 임팩트: LLM 파인튜닝 데이터 확보, 해석 품질 체계적 개선

**[INS-6] 소셜 레이어 — 까치 공유 카드**
- 벤치마크 근거: Co-Star의 SNS 공유 바이럴, 점신의 인맥보고서
- 사주까치 적용: 오늘의 운세 / 분석 결과 → 까치 마스코트 포함 공유 카드 생성 (이미지 다운로드 또는 링크 공유)
- 임팩트: 브랜드 인지도, 오가닉 유입

### P2 — 중기 적용 (수익화 + 성장)

**[INS-7] 코인/크레딧 단건 결제 도입 (포스텔러 벤치마크)**
- 벤치마크 근거: 포스텔러 전체 콘텐츠 30%만 유료, 코인 단건 결제로 구독 거부감 없이 수익화 → 2024년 100억 매출
- 사주까치 적용: 심층분석은 현재 로그인 필수 프리미엄 → 코인 1개 = 심층분석 1회, 코인 구매 또는 일일 광고 시청으로 무료 획득
- 임팩트: 비구독자도 결제 전환 가능, 수익원 다각화

**[INS-8] AI 해석 엔진 (Rule→LLM 변환, 현재 제품 방향 강화)**
- 벤치마크 근거: 포스텔러 "Posteller AI" 베타 출시, 운세박사 GPT ChatGPT 라이프스타일 10위, "챗GPT 사주" 검색량 3배 급증
- 사주까치 적용: 현재 Rule 엔진 → LLM 자연어 변환 (이미 제품 방향으로 설정됨) → 사용자 이름·나이·상황을 컨텍스트로 주입해 초개인화
- 임팩트: "와, 진짜 내 얘기네" 공감도의 핵심 무기, 경쟁사 대비 품질 우위

**[INS-9] 전문가 상담 마켓플레이스 (장기)**
- 벤치마크 근거: 점신 400명+ 전문가 전화상담 (분당 과금), Sanctuary 라이브 전문가, 사주나루 연매출 400억
- 사주까치 적용: 분석 결과 페이지 하단 "더 깊은 상담을 원하시나요? → 전문 역술가와 연결" CTA → 수수료 모델
- 임팩트: 고단가 수익원, 플랫폼 네트워크 효과
